# go-fastify

Simple server with fastify.

## How to start

```bash
npm i -g pnpm
pnpm i
pnpm dev
open http://localhost:4000
```

## How to build

```bash
pnpm build
pnpm start
```

## How to check code quality

```bash
pnpm check:all
```

## How to test

```bash
pnpm test
```

## Learn More

To learn Fastify, check out the [Fastify documentation](https://fastify.dev/docs/latest/).
